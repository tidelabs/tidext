Expand description

Signing transactions requires a Signer. This is responsible for providing the “from” account that the transaction is being signed by, as well as actually signing a SCALE encoded payload. Optionally, a signer can also provide the nonce for the transaction to use.

Some signers may fail, for instance because the hardware on which the keys are located has refused the operation.
